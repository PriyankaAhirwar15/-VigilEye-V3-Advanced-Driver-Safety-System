## рҹҺҜ What is VigilEye-V3?

**VigilEye-V3** is a production-grade, privacy-first AI safety system that monitors driver alertness in real time using only a standard webcam. It combines computer vision, behavioural analysis, and machine learning to detect fatigue before it becomes fatal вҖ” all without sending a single byte of data to the cloud.

```
рҹҡ—  Camera  в”Җв”Җв–ә  AI Analysis  в”Җв”Җв–ә  Fatigue Score  в”Җв”Җв–ә  Alert / Report
     рҹ“·           7 modules          0 вҖ“ 100             рҹ”Ҡ  рҹ“„  рҹ“Ҡ
```

**Why it matters:**
- Road accidents cause **1.35 million deaths** per year globally (WHO)
- **drowsy driving** accounts for **21%** of fatal crashes
- VigilEye-V3 runs on any laptop, **no specialised hardware required**

## рҹ§  Fatigue Scoring Algorithm

The heart of VigilEye-V3 is a **weighted multi-signal fatigue score** (0вҖ“100):

```
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ                  FATIGUE SCORE FORMULA                      в”Ӯ
в”Ӯ                                                             в”Ӯ
в”Ӯ  Score = (EAR Г— 0.35) + (PERCLOS Г— 0.30)                  в”Ӯ
в”Ӯ        + (MAR Г— 0.20) + (GAZE Г— 0.15)                     в”Ӯ
в”Ӯ                                                             в”Ӯ
в”Ӯ  Each signal is normalized to [0.0 вҖ“ 1.0] then Г—100        в”Ӯ
в”Ӯ                                                             в”Ӯ
в”Ӯ  Signal          Weight   Measures                          в”Ӯ
в”Ӯ  в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ в”Җв”Җв”Җв”Җв”Җв”Җ   в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ     в”Ӯ
в”Ӯ  EAR  (eye open)   35%   Eye closure speed + depth         в”Ӯ
в”Ӯ  PERCLOS           30%   % frames eyes closed / 60fr       в”Ӯ
в”Ӯ  MAR  (mouth)      20%   Yawning frequency + intensity     в”Ӯ
в”Ӯ  GAZE direction    15%   Head turn / look-away events      в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
```

### EAR Calculation (Eye Aspect Ratio)

```
         P2 в”Җв”Җв”Җ P3
        /         \
  P1 в”Җв”Җ             в”Җв”Җ P4
        \         /
         P6 в”Җв”Җв”Җ P5

  EAR = (вҖ–P2вҲ’P6вҖ– + вҖ–P3вҲ’P5вҖ–) / (2 Г— вҖ–P1вҲ’P4вҖ–)

  Normal open eye:  EAR вүҲ 0.30 вҖ“ 0.40
  Drowsy threshold: EAR < 0.22  в”Җв”Җв–ә alert triggered
```

### PERCLOS (Percentage Eye Closure)

```
  60-frame rolling window (вүҲ 2 seconds at 30fps)

  PERCLOS = (frames with EAR < 0.22) / 60 Г— 100%

  0%  в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ 100%
  в–Ҳв–Ҳв–Ҳв–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘  normal   (~5%)
  в–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘  drowsy   (>30%)
  в–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–‘в–‘в–‘в–‘в–‘в–‘в–‘в–‘  critical (>60%)
```

## рҹ”¬ Module Deep Dive

### `predictor.py` вҖ” Core Vision Engine

```python
# MediaPipe Face Mesh вҖ” 468 3D landmarks per frame
# Runs at 30fps on standard CPU hardware

EAR = (вҖ–P2вҲ’P6вҖ– + вҖ–P3вҲ’P5вҖ–) / (2 Г— вҖ–P1вҲ’P4вҖ–)   # Eye Aspect Ratio
MAR = (вҖ–P3вҲ’P7вҖ– + вҖ–P4вҲ’P8вҖ–) / (2 Г— вҖ–P1вҲ’P5вҖ–)    # Mouth Aspect Ratio

# PERCLOS: rolling 60-frame buffer
perclos_buffer.append(1 if EAR < 0.22 else 0)
PERCLOS = sum(buffer[-60:]) / 60 * 100
```

### `alcohol_detector.py` вҖ” 5-Signal Impairment Engine

```
Signal 1:  Eye Sway      вҖ” Low EAR mean + high EAR std deviation
Signal 2:  Head Sway     вҖ” High std deviation in gaze X/Y over 2 seconds
Signal 3:  Micro-sleep   вҖ” PERCLOS > 40% sustained
Signal 4:  Gaze Drift    вҖ” High variance in gaze direction buffer
Signal 5:  Yawn Spike    вҖ” > 4 yawns per minute

Trigger:   3 or more signals active simultaneously
Score:     (active_signals / 5) Г— 100, boosted if 3+ concurrent
```

### `night_mode.py` вҖ” CLAHE Enhancement

```
Input frame brightness:

  0  в”Җв”Җв”Җв”Җ 40       VERY DARK вҶ’ contrast Г—2.5, brightness +60, CLAHE
  40 в”Җв”Җв”Җв”Җ 80       DARK      вҶ’ contrast Г—1.8, brightness +40, CLAHE
  80 в”Җв”Җв”Җв”Җ 180      NORMAL    вҶ’ no enhancement
  180 в”Җв”Җв”Җ 255      BRIGHT    вҶ’ no enhancement

CLAHE: Contrast Limited Adaptive Histogram Equalization
  clipLimit=3.0, tileGridSize=(8,8)
  Applied in LAB colorspace for perceptual accuracy
```

### `face_recognition_module.py` вҖ” Offline Driver ID

```
Training:  OpenCV LBPH (Local Binary Pattern Histograms)
           100% offline вҖ” no cloud, no dlib, no DeepFace

Workflow:
  Register  вҶ’ capture face вҶ’ save JPEG вҶ’ retrain LBPH model
  Recognize вҶ’ run every 10th frame вҶ’ LBPH predict вҶ’ conf < 80 = match

Confidence:  LBPH confidence score (lower = better match)
             Display: 100 - raw_conf  (higher = more confident)
```
